IWP COMMUNITY CONNECTIONS V25 BROWSER REGRESSION

HOW TO RUN
1. Extract this entire ZIP to a normal folder on your computer.
2. Double-click RUN-BROWSER-REGRESSION.bat.
3. Let the Chrome test window open and close by itself.
4. The HTML results page opens automatically when testing finishes.

FIRST RUN
The runner may install its local browser testing engine. This can take a few minutes once.
Node.js LTS must be installed. The runner will tell you if it is missing.

ORGANIZER TEST SETUP
If organizer tests show a warning, run:
browser-tests\Setup-V21-Organizer-Profile.bat
Sign into the approved organizer Google account in the dedicated Chrome window, verify Organizer Access appears, then close only that dedicated window.

SAFETY
The suite is read-only. It does not create, publish, register, upload, email, or delete data.

REPORT LOCATION
browser-tests\reports\latest-report.html
