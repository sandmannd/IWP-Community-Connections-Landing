# IWP Community Connections V23 Release Notes

V23 completes Community Memories and delivers organizer analytics, a premium Adventure Details experience, a stronger registration flow, and improved public usability.

## Community Memories
Members can share photos after completed adventures. Organizers can upload, review, approve, hide, feature, and delete memories. Public galleries include responsive viewing, captions, featured states, empty states, keyboard controls, upload limits, and moderation safeguards.

## Organizer analytics
The Command Center now summarizes registrations, participants, group size, check-in rate, returning participants, Community Memories, top adventures, and six-month trends. Analytics can be exported to CSV.

## Public experience
Adventure Details uses clearer time, location, cost, journey, organizer, registration, and memory sections. Public discovery includes stronger keyboard support, focus states, reduced-motion behavior, and long-list performance improvements.

## Registration
Registration now provides live capacity guidance, event-aware payment choices, a final review, exact guest-name validation, paired emergency contacts, and persisted guest names.

## Deployment
This package is a release candidate until the deployed backend and browser regression suites complete with zero failures and zero warnings.
